# FiscoBot Togo

Assistant fiscal IA spécialisé CGI Togo 2025 · LPF · OHADA · SYSCOHADA

## Déploiement sur Vercel

1. Importer ce repo sur GitHub
2. Connecter à Vercel
3. Ajouter variable d'environnement : `ANTHROPIC_API_KEY`
4. Déployer

## Variables d'environnement requises

- `ANTHROPIC_API_KEY` : Votre clé API Anthropic (https://console.anthropic.com)
